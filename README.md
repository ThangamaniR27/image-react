# 📸 React Image Gallery

A simple and responsive **Image Gallery application** built using **React.js**. It displays images in a clean grid layout with smooth user experience.

---

## 🚀 Features

- 🖼️ Image grid layout
- 🔍 Click to view image (optional feature)
- 📱 Fully responsive design
- ⚡ Fast performance using React
- 🎨 Clean UI

---

## 🛠️ Tech Stack

- ⚛️ React.js
- 🧩 JavaScript (ES6+)
- 🎨 CSS / Tailwind CSS
